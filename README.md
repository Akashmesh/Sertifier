# Sertifier
Project Sertifier is an enterprise-grade solution designed to facilitate the design, distribution, and tracking of digital credentials with ease. This industry-based project comprises two main modules: Admin and Manager. The Manager module allows managers to register employees for seminars or workshops. Once an employee is registered, the Admin module enables the administrator to approve the registration or issue ID cards. Additionally, the admin can generate QR codes containing the employee's basic information and issue performance certificates based on grades.
The project is developed using JSP, Java Servlets, HTML, CSS, and JavaScript.
